Installationsanleitung für die "Now Playing"-Integration

4. Einrichtung der HTML-Datei in OBS

  1. Öffnen Sie OBS Studio.

    2. Diese Anleitung führt Sie durch die Installation und Einrichtung der "Now Playing"-Integration. Diese besteht aus drei Dateien: einem Tampermonkey-Skript, einer HTML-Datei für die Anzeige in OBS, und einem Node.js-Server-Skript.

    1. Benötigte Dateien

    • Tampermonkey-Skript: Tampermonkey Now Playing Youtube-1.0.24.user.js
    • HTML-Datei: now_playing.html
    • Server-Skript: server_ohne.js

    2. Installation des Tampermonkey-Skripts

    1. Installieren Sie die Browser-Erweiterung Tampermonkey, falls noch nicht geschehen.
    2. Öffnen Sie die Datei Tampermonkey Now Playing Youtube-1.0.24.user.js und fügen Sie das Skript in Tampermonkey hinzu.
    3. Aktivieren Sie das Skript. Dieses Skript sammelt die aktuellen Videoinformationen von YouTube und sendet diese an den Server.

    3. Einrichten des Node.js-Servers

    1. Stellen Sie sicher, dass Node.js auf Ihrem System installiert ist. Wenn nicht, laden Sie es von nodejs.org herunter und installieren Sie es.
    2. Speichern Sie das Skript server_ohne.js auf Ihrem Computer.
    3. Öffnen Sie ein Terminal (oder die Eingabeaufforderung) und navigieren Sie zu dem Ordner, in dem sich server_ohne.js befindet.
    4. Führen Sie im Terminal den Befehl npm init -y aus, um eine neue package.json zu erstellen.
    5. Installieren Sie die erforderlichen Abhängigkeiten mit folgendem Befehl:
      npm install ws
    6. Starten Sie den Server mit dem folgenden Befehl:
      node server_ohne.js
    7. Der WebSocket-Server läuft nun und ist bereit, Daten von YouTube zu empfangen.

    4. Einrichtung der HTML-Datei in OBS

    1. Öffnen Sie OBS Studio.
    2. Fügen Sie eine neue Browser-Quelle hinzu.
    3. Verweisen Sie in den Einstellungen der Browser-Quelle auf die Datei now_playing.html, die auf Ihrem lokalen Computer gespeichert ist.
    4. Stellen Sie die Abmessungen der Quelle auf die Größe ein (1400x500 Pixel).
    5. Speichern Sie die Einstellungen. Die "Now Playing"-Informationen werden nun in Ihrem Stream angezeigt, sobald der Server die Daten erhält.

    5. Funktionsweise

    Der Node.js-Server empfängt über das Tampermonkey-Skript die aktuellen Videoinformationen von YouTube (Titel, Kanal, Cover und Fortschritt). Diese Daten werden dann an die HTML-Datei weitergegeben, die in OBS angezeigt wird. Die Anzeige wird alle paar Minuten gemäß der Länge des Videos aktualisiert und für 30 Sekunden eingeblendet.

    6. Fehlerbehebung

    Falls Sie auf Probleme stoßen, stellen Sie sicher, dass:

    • Der Node.js-Server ordnungsgemäß läuft (keine Fehlermeldungen im Terminal).
    • Das Tampermonkey-Skript aktiviert ist und auf YouTube funktioniert.
    • Die OBS-Browser-Quelle korrekt auf die now_playing.html verweist.
